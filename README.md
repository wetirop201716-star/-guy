## Описание проекта

Графическое приложение (GUI) для конвертации валют. Использует внешний API для получения актуальных курсов валют и сохраняет историю операций в локальный файл.

## Технологии

*   **Python 3.x**
*   **Tkinter** — для создания графического интерфейса.
*   **Requests** — для работы с API.
*   **JSON** — для сохранения истории.
*   **Git** — система контроля версий.

### 2. Получение API-ключа

Приложение использует сервис [ExchangeRate-API](https://www.exchangerate-api.com/).
1.  Перейдите на сайт и создайте бесплатный аккаунт.
2.  В личном кабинете скопируйте ваш **API Key**.
3.  Откройте файл `main.py`.
4.  Замените значение переменной `API_KEY` на ваш ключ:
    ```python

    ```

## Использование интерфейса

1.  **Сумма**: Введите числовое значение в поле «Сумма».
2.  **Валюта «Из»**: Выберите валюту из первого выпадающего списка (например, `USD`).
3.  **Валюта «В»**: Выберите целевую валюту из второго списка (например, `RUB`).
4.  Нажмите кнопку **«Конвертировать»**.
5.  Результат появится во всплывающем окне и будет добавлен в таблицу истории внизу экрана.

## Сохранение истории

История всех успешных операций автоматически сохраняется в файл `history.json` в папке проекта. При следующем запуске приложения эта история будет загружена и отображена в таблице.

## Примеры использования (Тесты)

*   **Корректный ввод**: Ввод суммы `100`, выбор валют `USD -> RUB`. Ожидаемый результат: отображение курса и суммы в рублях.
*   **Некорректный ввод (отрицательное число)**: Ввод суммы `-50`. Ожидаемый результат: сообщение об ошибке «Сумма должна быть положительной».
*   **Некорректный ввод (ноль)**: Ввод суммы `0`. Ожидаемый результат: сообщение об ошибке «Сумма должна быть положительной».
*   **Одинаковые валюты**: Выбор `USD -> USD`. Ожидаемый результат: сообщение об ошибке «Выберите разные валюты».
